# Install Vibe Prospecting in Claude Code

Vibe Prospecting is available in the Claude plugin store. Install it with:

```text
/plugin install vibe-prospecting@claude-plugins-official
```

Restart Claude Code if the plugin or skill does not appear immediately.

## After install

You **must** follow the [Claude Code platform guide](../skills/vibe-prospecting/platforms/claude-code.md) for auth, workflow, and usage.
