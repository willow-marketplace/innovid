# Install Vibe Prospecting in OpenClaw

Install the Vibe Prospecting plugin from ClawHub:

```bash
openclaw plugins install clawhub:vpai-plugin
```

Restart OpenClaw if the plugin or skill does not appear immediately.

## After install

You **must** follow the [OpenClaw platform guide](../skills/vibe-prospecting/platforms/openclaw.md) for auth, workflow, and usage.
