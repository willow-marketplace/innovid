# Agent Kit Catalog Provenance

This directory is generated by the `endor-labs-agent-kit` publish workflow.

- `agent-kit-catalog.intoto.json` is the deterministic in-toto/SLSA-style
  statement emitted by `endor-agent-kit provenance-statement`.
- `manifest.sha256` records the checksum of the Agent Kit `manifest.json`
  that anchors all generated artifact checksums.

Do not edit these files by hand. Update Agent Kit source, regenerate,
and let the publish workflow open a new distribution PR.
