## How to Partner with AWS

Learn about the essentials of how to build, market and sell your solutions with the AWS Partner Network.

## Course Details

This learning path consists of 7 modules covering the AWS Partner Network (APN) and how startups can effectively partner with AWS.

### 1. AWS Partner Network (APN) Overview

An introduction to the AWS Partner Network and its benefits for startups.

### 2. APN Customer Engagements

Learn how to engage with customers through the AWS Partner Network.

### 3. Leveraging the Foundational Technical Review

Understand how to use the Foundational Technical Review to validate and improve your solutions.

### 4. Get Started on AWS Marketplace

Learn how to list and sell your solutions on AWS Marketplace.

### 5. AWS Global Startup Program

Overview of the AWS Global Startup Program and its resources for startups.

### 6. AWS Global Startup Program: Best Practices for Co-Marketing with AWS

Learn best practices for marketing your solutions alongside AWS.

### 7. AWS Global Startup Program: Best Practices for Co-Selling with AWS

Learn best practices for selling your solutions together with AWS.
