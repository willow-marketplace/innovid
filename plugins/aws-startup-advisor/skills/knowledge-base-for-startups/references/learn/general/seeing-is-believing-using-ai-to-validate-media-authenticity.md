## Seeing is believing: Using AI to validate media authenticity

## Overview

Generative AI has the power to create high fidelity images, video, and audio, limited only by imagination. But, doctored photos, deepfake videos, and voice cloning pose significant ethical risks. From scammers posing as an extended family member to fake political ads, there's a crisis of reality looming unless we take action.

Hear from the founders of DeepMedia, an AI communications company, on how they're using AI to build deep fake detection software to dispel false images and their implications. We'll explore real-life examples of how detection software helps validate the authenticity of media—and how we can create a future where what we see and hear is real, not replicant.
