## Startup Build Solutions

Searchable index of sample architectures and solution guides for startups building on AWS. Each detail file has an Overview, AWS services used, a metadata chip set (deployment method / estimated time / cost calculator), and a primary Get-Started link — typically an AWS Workshop or a GitHub sample repo.

Use the Keywords column to filter by category (GenAI, Container, Analytics, Security, Operation) or AWS service.
