## AWS Startups & Activate FAQ

## Frequently asked questions

Welcome to the new AWS Startups website. Find answers to frequently asked questions (FAQs) about the new website, AWS Activate program, credits and Showcase.

## Profile and Membership

### How do I sign in?

Navigate to the top of the AWS Startups website to sign in or create an account. Create an AWS Builder ID, then complete the sign up form to become an AWS Activate member.

### What is an AWS Builder ID?

AWS Builder ID is a new personal profile for every individual that builds on AWS. It provides access to tools and builder services on AWS, and your Builder ID stays with you as you move between jobs, schools, or other organizations. To learn more, visit [Sign in with AWS Builder ID](https://docs.aws.amazon.com/signin/latest/userguide/sign-in-aws_builder_id.html).

### How is an AWS Builder ID different than an AWS account?

Your AWS Builder ID is separate from any AWS account or sign in credential. You can use the same email for your AWS Builder ID and for the root user email of an AWS account.

An AWS account is a resource container with contact and payment information. It establishes a security boundary in which to operate billed and metered AWS services, like S3, EC2, or Lambda. Account owners can sign in to an AWS account in the AWS Management Console. To understand different AWS credentials, visit the [AWS sign-in user guide](https://docs.aws.amazon.com/signin/latest/userguide/differences-aws_builder_id.html).

### What is AWS Activate?

AWS Activate is the flagship startup program from AWS, the world’s most broadly adopted cloud. From the earliest stages, Activate provides founders and their teams with cloud credits, technical support, and business mentorship. Free to join, whether a founder is just starting to ideate an MVP or is on track to IPO, AWS Activate is a startup’s first step to prove that their world-changing ideas are possible.

### How do I become an AWS Activate member?

Joining Activate is easier than ever. First, you’ll enter basic profile information to create an [AWS Builder ID](https://docs.aws.amazon.com/signin/latest/userguide/sign-in-aws_builder_id.html). Once your Builder ID is created, you’ll complete a one-time membership form to help us better understand the needs of you and your startup.

## AWS Activate Credit Application

### Who can apply for AWS Activate Credits?

AWS Activate Credits applications are open to bootstrapped, self-funded, and funded startups up to and including Series A. Startups must have been founded within last 10 years and have a functioning website or public profile. Unfortunately, Startups building on AWS China Regions are not eligible for AWS Activate. If you are building on AWS China Regions, [please apply for the AWS Activate China program here](https://www.amazonaws.cn/en/startups/).

### I do not have all the application information right now. Can I come back later to fill and submit my application?

Signing in with your AWS Activate account allows you to start and stop your application as needed. Simply save a draft of your application, and your membership will save your place at the last step you completed.

### I received an email from AWS Activate that my Activate Provider has invited me to apply for the Activate Credits. What do I do next?

Activate Providers can invite their portfolio companies to apply for AWS Activate Credits via email. The link is personalized and speeds up the application process by pre-validating your relationship with the provider.****

**Important:** You must apply using the same email address that received the invitation from your Provider. For example, if the invite was sent to [john@appleseed.com](mailto:john@appleseed.com), you must submit your application using [john@appleseed.com](mailto:john@appleseed.com) for the pre-validation to work correctly.

### What happens if I don't use my email invite?

Email invitations expire after 30 days. If your invitation has expired, contact your provider directly and ask them to resend it. Providers can send a new invitation through Provider Central in just a few clicks.

### Should I ask my provider to send me their Org ID or an email invite?

Email invitations are recommended. They streamline the approval process by pre-validating your relationship with the provider and typically result in faster approvals. Applications submitted through email invites are approved within 3-5 days compared to 7-10 days for applications using Org ID. For the fastest path to receiving AWS Activate Credits, request an email invitation from your provider.

### What is an Organization ID (Org ID)?

Each AWS Activate Provider (venture capital firm, accelerator, incubator, or other startup-enabling organization) has a unique identifier called an Org ID. An Org ID is confidential and should not be shared - it is not a promotional code. To apply for Activate Portfolio, you must enter the Org ID given to you by your AWS Activate Provider during the application process. If you are associated with an AWS Activate Provider, but you do not have their Org ID, please reach out to your AWS Activate Provider to request it. If you are not associated with an AWS Activate Provider, you may be eligible to apply for AWS Activate Founders.

### Can I apply multiple times with the same Org ID?

No, an Org ID can only be used one time. If you would like to apply for additional AWS Activate Credits, you need to get a different Org ID from an Activate Provider.

### I’m associated with more than one AWS Activate Provider. Can I apply for AWS Activate Credits more than once?

Yes. You can apply for AWS Activate Portfolio Credits more than once as long as 1) each application is for a greater amount of AWS Activate Credits than your previously approved application and 2) you haven’t received the maximum lifetime value in AWS Activate Credits (not including AWS Support plan credits).

### How many times can I benefit from a specific AWS Activate Provider’s offer?

You can only benefit from an AWS Activate Provider's specific offer once. If your Activate Provider has two offers, you may apply for the second offer as long as 1) it’s for a greater amount of AWS Activate Credits than your previously approved application and 2) you haven’t received the maximum lifetime value in AWS Activate Credits (not including AWS Support plan credits).

### My startup has received AWS credits previously from AWS Activate Portfolio. Am I eligible to apply again?

You can apply for AWS Activate Credits again as long as 1) your application is for a greater amount of AWS Activate Credits than your previously approved application; 2) you haven’t received the maximum lifetime value in AWS Activate Credits not including AWS Support plan credits; 3) you have a valid Org ID; and 4) you still meet the criteria of the Activate Credits program.

### Can I submit more than one application at the same time?

No. You can only submit one application at a time. If you submit two AWS Activate Credit applications at the same time for different credit offers, the application with the smaller credit amount will be automatically rejected.

### I am a founder of two startups. One of my startups has already received AWS Activate Credits. Can I apply again for my second startup?

Yes. Please note: each startup must have its own AWS account. If you apply from the same AWS account, even though the second startup is a separate entity, your application will be rejected.

### Why did I receive a different amount of AWS Activate Credits than what I applied for?

If your startup has previously received AWS Activate Credits from AWS Activate, and you are applying for additional credits which is a larger amount than what you have received thus far, you will receive the difference between the two AWS Activate Credit awards. For example, if you received $10,000 USD in AWS Activate Credits, and you subsequently applied for $100,000 USD in AWS Activate Credits, you will receive $90,000 USD in AWS Activate Credits. If you are applying for a smaller amount of AWS Activate Credits than what you have already received, or if you have already reached the maximum lifetime value of AWS Activate Credits, your application will be declined.

### I have received multiple AWS Activate Credit offers that vary in credit amounts from the same AWS Activate Provider e.g. $5,000 USD, $25,000 USD, and $100,000 USD. If I accept one, will that make me ineligible for the others?

You can apply for AWS Activate Portfolio more than once as long as 1) each application is for a greater amount of AWS Activate Credits than your previous approved application, 2) you haven’t received the maximum lifetime value in AWS Activate Credits, 3) you still meet the program criteria, and 4) have a valid Org ID. For example, if you apply for and receive $10,000 USD in AWS Activate Credits, you can apply for $100,000 USD in AWS Activate Credits as long as you are still affiliated with the Activate Provider at time of application. If approved you will receive $90,000 USD in AWS Activate Credits, bringing you up to $100,000 USD in AWS Activate Credits.

### I have received AWS Credits from AWS EdStart or other AWS Credit program. Does this make me ineligible for AWS Activate?

Yes, you are still eligible for AWS Activate Credits. AWS Activate Credits issued by Activate are not impacted by previous AWS Activate Credits you may have received through other programs, including AWS EdStart.

Benefits of AWS Activate Credits include:

- **Build on AWS with AWS Activate Credits:**if approved, AWS Activate Credits will be added to your AWS Account and will automatically apply towards the cost of AWS Eligible Services. AWS Activate Credits can also be used for third-party models on Amazon Bedrock.
- **Explore other startups and promote your startup at Startup Showcase:** AWS Activate members that have received credit offers can promote their startup on [Startup Showcase](https://aws.amazon.com/startups/showcase), a directory designed for AWS startup customers. If you’ve built and launched a product on AWS, create a page to promote your offerings to potential customers, investors, and partners.

- **Exclusive offers:** as you start building using your AWS Activate Credits, you can get ahead with Activate Exclusive Offers from organizations such as ZenDesk, Stripe, DataDog, Brex, HubSpot, and Chargebee.
- **Pre-built infrastructure templates:**AWS Activate Credit approval also allows you to explore solutions with [Build](https://aws.amazon.com/startups/build), a collection of cloud templates and integration guides curated by AWS experts to tackle the most common industry- and product-specific use cases.

### What can I use AWS Activate Credits for?

AWS is the world’s most comprehensive and broadly adopted cloud, offering over 200 fully featured services from data centers globally. You can use your AWS Activate Credits for AWS “Eligible Services,” from infrastructure technologies, such as compute, storage, and databases, to emerging technologies, such as machine learning, artificial intelligence, data lakes, analytics, and Internet of Things. AWS Activate Credits are also redeemable for third-party models on [Amazon Bedrock](https://aws.amazon.com/bedrock/), our fully-managed service that offers a choice of high-performing foundation models (FMs) from leading artificial intelligence (AI) companies, like [AI21 Labs](https://www.ai21.com/), [Anthropic](https://www.anthropic.com/), [Cohere](https://cohere.com/), [Meta](https://llama.meta.com/), [Mistral AI](https://mistral.ai/), [Stability AI](https://stability.ai/), and Amazon via a single API.

For information regarding Eligible and Ineligible Services, please visit the [AWS Promotional Credit Terms & Conditions](https://aws.amazon.com/awscredits/) and the [AWS Activate Terms & Conditions](https://aws.amazon.com/activate/terms/). Any service not listed in the [AWS Billing and Cost Management Console](https://console.aws.amazon.com/billing/home?#/account) is not covered by AWS Activate Credits.

### What’s not covered by AWS Activate Credits?

AWS Activate Credits cannot be applied to any fees or charges for Amazon Mechanical Turk, AWS Managed Services, Ineligible AWS Support, AWS Marketplace (with the exception of charges incurred through the use of third-party foundation models available on Amazon Bedrock), AWS Professional Services, AWS Training, AWS Certification, Amazon Route 53 domain name registration or transfer, any Services for mining for cryptocurrency, or any upfront fee for any Services such as Savings Plans and Reserved Instances. For information regarding Eligible and Ineligible Services, please visit the [AWS Promotional Credit Terms & Conditions](https://aws.amazon.com/awscredits/) and the [AWS Activate Terms & Conditions](https://aws.amazon.com/activate/terms/). Any other service not listed in the [AWS Billing and Cost Management Console](https://console.aws.amazon.com/billing/home?#/account) is not covered by AWS Activate Credits.

### How do I use my AWS Activate Credits?

No action is required from you to activate your AWS Activate Credits. AWS Activate Credits offset charges for AWS EligibleServices incurred during or following the billing cycle in which you receive the credits. You can view, track, and monitor your AWS Activate Credits in the [AWS Billing and Cost Management Console](https://console.aws.amazon.com/billing/home?#/account).

### Do AWS Activate Credits expire?

Yes, AWS Activate Credits have expiration dates. To view your AWS Activate Credits’ expiration date, please visit the [AWS Billing and Cost Management Console](https://console.aws.amazon.com/billing/home?#/account).

### Why do I need to upgrade to a Paid Tier Plan? How do I do it?

Free account plans are not eligible for Activate promotional credits. Find instructions on how to upgrade your account plan [here](https://docs.aws.amazon.com/awsaccountbilling/latest/aboutv2/free-tier-plans.html).

### Where can I find details about my AWS Activate Credits including credit usage and their expiration date?

You can find details about your AWS Activate Credits, including the exact credit amount, their expiration date, remaining balance, and the applicable AWS services, on the [AWS Billing and Cost Management Console](https://console.aws.amazon.com/billing/home?#/account). AWS Activate also informs startups when their credits are going to expire or exhaust via email: 1) credits are expiring (60 days prior expiration, 30 days prior expiration, expired) and 2) Account credits are exhausting (75% credits exhausted, 100% exhausted).

### How are my AWS Activate Credits applied against my bill? Can I use them to cover historical AWS bills?

AWS Activate Credits in your AWS account are applied only to offset AWS Eligible Services fees and charges incurred during or following the billing cycle in which you receive the credits. AWS Activate Credits cannot be used to offset or cover historical AWS bills. For example, if your AWS Activate Credits are issued in your AWS account in the month of May, your AWS Activate Credits will not count against your AWS bills from April or previous months.

### My AWS account is part of consolidated billing through a reseller. How do I make sure my Activate Credits are applied correctly?

To ensure your Activate Credits are applied correctly through consolidated billing through a reseller:

1. Check that only your company's accounts are linked in consolidated billing on the [AWS Console.](https://console.aws.amazon.com/)
2. Ask your reseller to confirm: a) your credits are isolated from other customers, and b) the main billing account prevents credit sharing.
3. Review your [AWS bill](https://console.aws.amazon.com/billing/home?#/) regularly to verify credit usage.

If you notice any issues with your credits, contact your reseller first, and then reach out to [AWS support](https://aws.amazon.com/contact-us/) if needed.

### How are my AWS Activate Credits applied to Reserved Instances or Saving Plans?

AWS Activate Credits are applied to Reserved Instances and Saving Plans the same way as other AWS Eligible Services. Upfront fees for Reserved Instances and Savings Plans are not covered by AWS Activate Credits.

### My credits are about to expire. Can you extend the expiration date of my AWS Activate Credits?

The AWS Activate team is unable to extend the expiration date of AWS Activate Credits, even if there are remaining credits left unused.

### Can my AWS Activate Credits be refunded or re-issued?

AWS Activate Credits cannot be refunded or re-issued into your AWS Account, even if your AWS Activate Credits were incorrectly used or accidentally consumed.

### My AWS Activate Credits were used by mistake. Can I get more credits?

Unfortunately, the AWS Activate team is unable to apply more AWS Activate Credits to your account.

### Can I transfer or sell my AWS Activate Credits?

No. AWS Activate Credits cannot be transferred into another AWS account. You may not sell, license, rent, or transfer AWS Activate Credits. If you violate this policy, your AWS Activate Credits will be revoked. For more information, please refer to the [AWS Promotional Credit Terms & Conditions](https://aws.amazon.com/awscredits/) and the [AWS Activate Terms & Conditions](https://aws.amazon.com/activate/terms/). If you are looking for information on how to share credits within your AWS accounts, please refer to [Applying AWS Credits](https://docs.aws.amazon.com/awsaccountbilling/latest/aboutv2/useconsolidatedbilling-credits.html). For customers in India, please see [Consolidated Billing in India](https://docs.aws.amazon.com/awsaccountbilling/latest/aboutv2/useconsolidatedbilling-India.html).

### Can I share my AWS Activate Credits with associated accounts? How are AWS Activate Credits shared among parent and child accounts?

If you have enabled credit sharing in the [AWS Billing and Cost Management Console](https://console.aws.amazon.com/billing/home?#/account), your AWS Activate Credits may be used by the parent account as well as any associated child accounts. If you wish to turn on/off Credit sharing across your accounts, [please follow the instructions here](https://docs.aws.amazon.com/awsaccountbilling/latest/aboutv2/useconsolidatedbilling-credits.html#credit-sharing). Please note: The credit will apply to the account effective immediately when the credit sharing is enabled in the same month.

### I have a question about how AWS Activate Credits affect taxes on my bill. Who should I contact?

For billing or account inquiries, please [click here to sign in](https://console.aws.amazon.com/support/home#/case/create?issueType=customer-service) to your AWS account and submit your billing or account request, or contact [AWS Customer Support](https://aws.amazon.com/contact-us/).

## AWS Activate Providers

### What is an AWS Activate Provider?

An [AWS Activate Provider](https://aws.amazon.com/startups/lp/aws-activate-provider) is a venture capital firm, angel investor, accelerator, incubator, or other startup-enabling organization that has partnered with AWS Activate to provide Activate benefits to their affiliated startups.

### Why should our firm become an AWS Activate Provider?

AWS Activate is the flagship startup program from AWS, the world's most broadly adopted cloud. From the earliest stages, Activate provides founders and their teams with promotional credits, technical support, and business mentorship. Becoming an AWS Activate Provider allows your institution to provide AWS Activate Credits and specialized resources to your portfolio of startup companies. This helps attract and support innovative startups by giving them access to AWS cloud resources.

### What benefits do AWS Activate Providers receive?

AWS Activate Providers receive credit packages to distribute to eligible startups, access to AWS technical and business resources, and support from the Activate team. These benefits enable Activate Providers to offer enhanced value to their portfolio companies, supporting their growth and success on AWS.

### What is the eligibility criteria to become an Activate Provider?

AWS Activate accepts applications from venture capital firms and accelerator programs that are actively investing in startups. Note that Activate is not currently accepting applications from non-investing organizations such as co-working spaces.

### How can my firm apply to become an AWS Activate Provider?

AWS Activate is accepting applications from venture capital firms, seed investors, and accelerator programs that are actively investing. By becoming an Activate Provider, you can offer Activate Credits and specialized resources to your portfolio companies, helping them build and scale on AWS. To apply, first create your Builder ID and fill out your AWS Activate profile. Next, you will be prompted to complete the application and provide information about your institution and investment portfolio. Get started [here](http://aws.amazon.com/startups/lp/aws-activate-provider). Once the application has been submitted, it will be reviewed within 7-10 business days.

### Can our firm request additional Activate Credits?

AWS evaluates Activate Providers annually to determine credit package eligibility and is not currently accepting requests for changes to these packages.

### Can our portfolio companies apply for additional AWS Activate Credits?

After receiving initial AWS Activate Credits, startups can seek additional credits by either connecting with their AWS account team or applying through Org IDs from other eligible providers in their network.

### Why are my portfolio companies getting rejected for Activate Credits even though they're approved for the portfolio?

In addition to being part of your approved portfolio, startups must qualify for AWS Activate Credits based on specific eligibility criteria. Criteria includes factors like startup age, funding status, and previous credit usage.

### What steps can I take if my portfolio company's application was rejected?

If a startup's application is rejected, review the reasons provided and verify that the startup meets all eligibility criteria. If you need additional assistance, please reach out to AWS Activate support on the [Contact Us](https://aws.amazon.com/startups/contact-us) page.

### How can I update my firm's contact information as an existing Activate Provider?

Please request an update through [Contact Us](https://aws.amazon.com/startups/contact-us).

### What is AWS Activate Provider Central?

AWS Activate Provider Central is where venture capitalists, accelerators, and other startup enabling organizations who are Activate Providers can manage their portfolio companies. It enables you to view and share your Organization ID, invite startups through one-time, pre-validated referral links for faster applications processing and approvals, track portfolio companies' credit applications for the past 24 months, and access resources to help share the Activate program with startups. It can be accessed [here](https://aws.amazon.com/startups/providers).

### How do I join Provider Central?

Existing Activate Providers will receive an email invitation from the AWS Activate team with an invite link to join Provider Central. Click the link in your invitation email and follow the setup instructions to open your account. Your existing organization ID and portfolio data will be automatically populated in the centralized hub. If you are interested in becoming an Activate Provider, please apply [here](https://aws.amazon.com/startups/providers). Newly approved providers will receive a unique invitation link to access Provider Central in their acceptance email.

### Will my existing Organization ID (Org ID) change?

No, your Organization ID will remain the same. Startups in your portfolio can continue using your existing Org ID to apply for AWS Activate Credits without any changes. All existing processes and links will continue to work. Additionally, Provider Central now enables you to refer your startups with just 3 clicks: login, enter startup email, and send invite making the referral process even easier while maintaining all your current capabilities.

### How does Provider Central simplify inviting startups to join AWS Activate?

As a provider, you can now refer your startups with just 3 clicks: login, enter the startup’s email, and send the invite. The pre-validated referral link cuts approval time from 8 days to 3 days, streamlining the process for you and your portfolio companies.

### What if I have trouble joining Provider Central using my invitation link?

If you have trouble joining Provider Central using your invitation link, first verify you are using the same email address that received the invitation. Next, check the date of your invitation email, as links expire after 30 days. If these initial checks don't resolve the issue, try these troubleshooting steps: 1) use an incognito or private browser window to eliminate browser extension interference, 2) clear your browser cache and cookies, and 3) copy and paste the complete invitation URL into your browser address bar. If you continue experiencing issues after trying these solutions, contact AWS Activate support through the Contact Us [[https://aws.amazon.com/startups/contact-us/contact-support](https://aws.amazon.com/startups/contact-us/contact-support)] and include the specific error message you are seeing.

## Personalized Journey

### What is my personalized journey?

Your personalized journey is a custom set of tools, resources, and trainings on the AWS Startups website designed to help startups launch and scale on AWS. This guided plan is available to all startups who have applied for AWS Activate Credits. The journey is tailored specifically to your needs based on your profile and AWS Activate Credits application, providing you with guides and tools to help you get started, build your product, and grow your business on AWS. Whether you're just starting out or looking to expand, this personalized plan offers valuable resources to support your startup's journey on AWS.

### How do I access my personalized journey?

To access your personalized journey, start by creating an AWS Builder ID account and completing your Activate profile. Then, apply for AWS Activate Credits. Your custom plan will be automatically created once you submit your credit application. You can view a snapshot of your plan on the AWS startups homepage widget or access the full plan directly at [aws.amazon.com/startups/your-journey](http://aws.amazon.com/startups/your-journey). To stay informed about your journey, you can opt in for monthly digest emails. You can easily opt out at any time on your plan page.

### What resources are included in the plan?

Your personalized plan is designed to help you leverage Activate benefits and build your startup using AWS services. The plan is organized into four key milestones: 1) Getting started with AWS, 2) Defining your MVP, 3) Building your MVP, and 4) Launching your product. ## Migration Capability

### What is the new migration capability on the AWS Startups experience?

A new AI-powered capability that generates a personalized migration plan from GCP, OpenAI, or Gemini to AWS, then helps you execute it. Get a plan quickly, including service mapping, cost estimates, architecture diagrams, Terraform templates, credit eligibility, and a step-by-step runbook. Then choose the execution path that fits your workload: AI-agent guided, work with an AWS expert, or connect with a certified AWS Partner.

### Which cloud providers and workloads are supported?

The new migration capability supports Google Cloud Platform (GCP) infrastructure migrations - including, for example, Kubernetes to EKS/ECS/Fargate, PostgreSQL/MySQL to RDS/Aurora, and Google Cloud Storage to S3, as well as AI/LLM inference migrations from OpenAI or Gemini to Amazon Bedrock.

### How much does the migration capability on AWS Startups cost?

There is no charge for using the migration capability. There are no usage limits for generating migration plans. You only pay for the AWS resources you provision during your actual migration. If you choose the AI-Agent Guided path, you select your own inference provider (Amazon Bedrock, OpenAI, or Anthropic). LLM inference costs are billed by whichever provider you choose.

### How do I generate my migration plan?

You can pick the experience that fits your workflow:

1. **Web Experience —** Start from your browser at aws.amazon.com/startups/migrate/wizard. Answer a few questions, upload relevant data, and get a complete migration plan.
2. **IDE Plugin —** Generate your migration plan without leaving your IDE. Available for Kiro, Claude Code, and Cursor. When you're ready to execute, your plan routes to the migration capability on AWS Startups web experience.

### How long does it take to generate a plan?

Generating a migration plan typically takes under 15 minutes, though larger or more complex codebases may require additional processing time. Actual migration timelines depend on your infrastructure complexity and readiness.

### What's included in my migration plan?

Architecture recommendations, cost estimates, infrastructure diagrams, deployment runbooks, IaC templates, and an AI migration guide (if applicable).

### Am I guaranteed to receive the credit amount displayed?

No. The credit amount shown is based on your estimates and the information you provided during the assessment. Final eligibility will be determined by AWS based on standard eligibility and verification checks.

### How do I execute the migration plan?

Once your plan is generated, choose the execution path that fits your workload. You can always change your mind later.

**AI-Agent Guided:** Run the migration with the AI agent directly from your IDE (Kiro, Claude Code, or Cursor). The agent handles provisioning, data migration, configuration updates, and testing. You confirm key decisions at each step. Migrate your full workload or start with a proof of concept (POC).

**Work with AWS Experts:** Share your migration plan with an AWS expert. Get guidance on architecture decisions, edge cases, and execution.

**Work with an AWS Partner:** Get matched with a certified AWS Partner. They handle execution end-to-end.

Your migration includes intelligent service recommendations, cost comparison, Terraform deployment templates, a phased migration runbook, architecture diagrams, and guided deployment support at every step.

### How does the AI Agent guided migration agent work?

You download your migration plan and install the agent via your terminal (macOS Terminal or Windows PowerShell). Once you configure your AWS credentials and select a foundation model through Amazon Bedrock as your inference backend, the agent uses it to reason through and execute your migration tasks: provisioning infrastructure, setting up data transfer, migrating storage, updating configurations, and running post-migration tests.

### Are there any prerequisites to using the AI Agent guided execution path?

Yes. You need Docker Desktop running on your machine and an active AWS account before you can install and use the migration agent.

### What if I get stuck during AI Agent guided execution?

The migration capability provides progress tracking and automated blocker identification throughout AI Agent guided execution. If you get stuck, you can switch to working with an AWS specialist or an AWS Partner.

### What if I need hands-on help?

Click "Request Support" from your plan to connect with an AWS specialist or an AWS Partner.

### What if my AWS Partner doesn't respond?

If you haven't heard back from your selected AWS Partner, please let us know through Contact Us. In the meantime, you can reach out to another Partner from the list, or connect with an AWS expert who can also pair you with an expert to keep your migration moving.

### How does AWS secure my data?

The new migration capability uses your data only to generate your migration plan. Any code you provide is deleted after 180 days, and all data is encrypted in transit.

### Is my code used to train AI models?

No. Your data is used solely to generate your migration plan and is not used for model training.

### Can I delete data that I upload?

Yes. You can delete your data and individual plans, and all uploaded files are automatically removed after 180 days.

### Why can't I upload my file?

Check that it's a .csv (billing) or .zip (code), isn't password-protected, and is within the size limit.
