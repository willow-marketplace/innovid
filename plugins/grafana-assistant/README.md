# Grafana Assistant

Cursor plugin with skills and rules for developing and using the Grafana Assistant app and CLI.

## Included components

- `rules/grafana-assistant.mdc` — best practices for using Grafana MCP tools effectively
- `skills/grafana-assistant-cli/SKILL.md` — skill for using the `grafana-assistant` CLI to interact with Grafana Assistant via A2A API

## Related

For the Grafana MCP server (40+ tools for dashboards, alerting, Prometheus, Loki, etc.), install the **grafana-mcp** plugin separately.
